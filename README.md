# web-baccarat-notes

## 项目简介

本仓库用于归档与发布多个独立的 HTML 页面。这些页面主要围绕特定主题整理而成，以静态文档的形式存放，不针对任何单个域名或具体网站。仓库内容仅供学习与参考，不构成任何形式的服务或产品。

## 目录说明

```
├── README.md            # 本文件
├── pages/               # 存放所有 HTML 页面
│   ├── index.html       # 页面索引（可选）
│   └── ...
└── assets/              # 公共资源（如样式、脚本、图片）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：所有 HTML 页面均放置在此目录下，可按主题或时间进一步细分。
- `assets/`：存放页面共用的样式表、JavaScript 脚本及图片等资源文件。

## 页面归档说明

每个 HTML 页面均为独立完整的文档，可直接在浏览器中打开运行。页面之间无强依赖关系，部分页面可能引用公共资源（位于 `assets/` 目录下）。归档内容会不定期更新或调整，恕不另行通知。
